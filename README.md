## What This Does
Automatically saves incoming Gmail emails 
to Google Sheets using n8n automation.

## Tools Used
- n8n
- Gmail
- Google Sheets
- OAuth2

## How It Works
1. Gmail Trigger detects new email
2. Data extracted from email
3. Row added to Google Sheets automatically
